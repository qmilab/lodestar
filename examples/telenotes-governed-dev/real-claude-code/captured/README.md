# Captured evidence

This directory holds the artifacts from a **real Claude Code run** against the
Telenotes proxy:

- `report.md` — the `lodestar report` output for the session (see
  `../RECIPE.md` → "Render the trust report").
- `transcript.md` — the notable turns of the Claude Code session.

It is intentionally empty until a real run is performed. The evidence is not
generated by CI and is **not** synthesised — it must come from an actual
interactive Claude Code session, captured by hand, with the Claude Code version
and date noted (the permission model that forces MCP-only tool use can change
between versions).

Until then, the deterministic `../scripted-run/` and `../poison-run/` (with
their committed reports under `../reports/`) are the reproducible evidence that
the governed pipeline works end-to-end.
